# Personal_Finance_Manager_App
● This is a React web app that helps the user keep track of their financial expenses.

● In order to run this app, you need npm and you need to run this command to get chart.js (if you don't already have it): 

npm install chart.js

● You can start the development server to see the web app in action!

Steps:
1. Type the following command in the command line:
   npm start
3. Type the following url in a web browser:
   http://localhost:3000/
